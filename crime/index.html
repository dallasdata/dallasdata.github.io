Crime

All crimes reported to the Dallas Police Department since 2001.

This data is compiled from data obtained via the Dallas Police Department website (for all data prior to June, 2014) and the Dallas Open Data Portal (for all data since June, 2014). Data prior to June, 2014 has a number of consistency issues which we are attempting to clean up, but in the meantime it's provided as-is.

Get the data

The data is available as downloadable CSV files, with one file per year, named by year. They are all available under the http://dallas-data.s3.amazonaws.com/crime/ URL hierarchy. For example, /crime/2001.csv contains all crimes begun in the year 2001.

For each crime, the following fields are defined:

start_date_year
Year that the crime began.
start_date_month
Month that the crime began, starting from 1 (January).
start_date_day
Day of the month that the crime began, starting from 1.
start_time_hour
Hour that the crime began, starting from 0.
start_time_min
Minute that the crime began, starting from 0.
end_date_year
Year that the crime ended.
end_date_month
Month that the crime ended, starting from 1 (January).
end_date_day
Day of the month that the crime ended, starting from 1.
end_time_hour
Hour that the crime ended, starting from 0.
end_time_min
Minute that the crime ended, starting from 0.
report_date_year
Year that the crime was reported.
report_date_month
Month that the crime was reported, starting from 1 (January).
report_date_day
Day of the month that the crime was reported, starting from 1.
report_time_hour
Hour that the crime was reported, starting from 0.
report_time_min
Minute that the crime was reported, starting from 0.
street_number
Numerical street number for the location at which the crime occurred.
street
Street name for the location at which the crime occurred.
apartment
Numerical apartment number for the location at which the crime occurred.
city
City for the location at which the crime occurred.
state
State for the location at which the crime occurred.
zipcode
Zipcode for the location at which the crime occurred.
offense_description
Short description of the crime.
